1. THE SYSTEM CONCEPT
---------------------
Nginx is deployed ahead as a Reverse Proxy Gatekeeper. It interfaces 
with external incoming traffic on high-level ports (Port 8080) and 
routes payload data safely back into private internal container sockets (Port 5000).

2. CONFIGURATION PAYLOAD BLOCK
------------------------------
server {
    listen 8080 default_server;
    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
